# Termux Files
## Android SDK Setup & Utilities
The bash scripts in this repo are compiled from various sources on GitHub - for easier and more convenient use for the Termux terminal. The original developers are:

* **01.android.sdk.setup.sh**: [Sohil876/termux-sdk-installer](https://github.com/Sohil876/termux-sdk-installer)
* **02.android.sdk.setup.sh**: [Keyaru-code/android-sdk-termux](https://github.com/Keyaru-code/android-sdk-termux)
* **03.android.sdk.setup.sh**: [illegalvoid/termux-build-apps](https://github.com/illegalvoid/termux-build-apps)
* **04.android.sdk.setup.BASH.sh**: [ahksoft/termux-android-build-environment](https://github.com/ahksoft/termux-android-build-environment)
* **04.android.sdk.setup.ZSH.sh**: [ahksoft/termux-android-build-environment](https://github.com/ahksoft/termux-android-build-environment)
* **05.android.sdk.setup.sh**: [Willie169/termux-android-sdk-ndk](https://github.com/Willie169/termux-android-sdk-ndk)
* **06.android.sdk.setup.sh**: [MatejMagat305/vab-termux](https://github.com/MatejMagat305/vab-termux)
